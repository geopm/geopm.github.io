◆ set_dim()

void geopm::TensorTwoD::set_dim ( size_t  rows,
size_t  cols 
)

Set dimensions of 2D tensor.

Parameters
[in]rowsThe number of "rows" or 1D tensors
[in]colsThe number of "cols" or the size of each 1D tensor

If the instance contains more than rows 1D tensors, it will be truncated. If the instance contains fewer than rows 1D tensors, it will be expanded to a total number of rows, uninitialized. The individual 1D tensors will be managed similarly.

Exceptions
geopm::Exceptionif rows = 0 and cols > 0
