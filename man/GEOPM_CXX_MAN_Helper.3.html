NAME

geopm::Helper - common helper methods

SYNOPSIS

#include <geopm/Helper.hpp>

Link with -lgeopm (MPI) or -lgeopmpolicy (non-MPI)

template <class Type, class ...Args> std::unique_ptr<Type> make_unique(

Args &&...args);

std::string read_file(

const std::string&path);

std::vector<std::string> split_string(

const std::string &str,
const std::string &delim);

DESCRIPTION

The Helper.hpp file defines various utility functions.

make_unique()

Implementation of std::make_unique (C++14) for C++11, scoped to the geopm:: namespace. Note that this version will only work for non-array types.

read_file()

Reads the file specified by path and returns the contents in a string.

split_string()

Splits a string str according to a delimiter delim and returns the pieces in a vector. The delimiter cannot be empty.
